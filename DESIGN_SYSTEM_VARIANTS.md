# Vevhu Manager Dashboard — Modern Premium Design System (3 Variants)

## Variant 1: Corporate Command Center (Default Brand)
- **Primary Color**: `#F3772D` (Vevhu Warm Orange)
- **Accent**: `#1976D2` (Vevhu Corporate Blue)
- **Background**: Soft Tint `#F8FAFC` / Dark Slate `#0A0E0F`
- **Navigation**: Left Vertical Sidebar with glowing active indicator
- **Charts**: Recharts with smooth gradient fills and glass tooltips
- **Tables**: Interactive data grids with quick filter pills, column sorting, and instant CSV/PDF export.

## Variant 2: Midnight Executive Dark (High-Performance Dark Theme)
- **Primary Color**: `#38BDF8` (Sky Cyber Blue) + `#F97316` (Vibrant Amber)
- **Background**: OLED Obsidian `#0B0F17`
- **Cards**: Surface `#111827` with `#1F2937` borders
- **Charts**: Neon glowing line & bar charts with high-contrast tooltips
- **Tables**: Dark matrix data grids with status glowing badges.

## Variant 3: Clean Enterprise Studio (Minimalist Light Theme)
- **Primary Color**: `#0F172A` (Slate Black) + `#2563EB` (Bright Sapphire)
- **Background**: Crisp White `#FFFFFF`
- **Cards**: `#F8FAFC` with subtle `#E2E8F0` borders
- **Charts**: Minimalist monochrome & duo-tone metric graphs
- **Tables**: Clean borderless tables with subtle row hover effects.
